MinistryCard 배포 설정 패키지

1) 설정값 입력

2) config.js 생성 결과

3) Apps Script 상단 설정

4) Supabase 초기화 SQL

5) Google Sheets 시트 구성

  1. 새 스프레드시트 생성 후 아래 이름의 시트를 만듭니다.
  2. 첫 행 헤더를 정확히 입력합니다.
  3. Apps Script에 Code.gs 업로드 후 위 SHEET_ID를 적용합니다.
  4. 웹 앱 배포 후 URL을 config.js의 API_URL로 입력합니다.

6) 배포 순서

  1. Supabase SQL Editor에서 초기화 SQL 전체 실행
  2. Authentication에서 Email 비활성(선택), RLS 정책 확인
  3. Google Sheets + Apps Script 배포(Web app, 누구나 접근 가능)
  4. 생성된 config.js를 프로젝트 루트의 config.js에 반영
  5. 정적 호스팅 배포 후 동작 확인(로그인/구역/방문 저장)

서비스 키(service_role)는 클라이언트 파일에 넣지 않습니다.
